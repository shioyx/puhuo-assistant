# 1688铺货助手Pro

> 1688采集 → 编辑优化 → 违禁词检测 → 一键铺货抖音小店

## 功能特性

- 🔍 **内嵌1688浏览器** - 不离开工具页面，直接在内嵌浏览器里搜货、鼠标悬停一键采集
- 📦 **采集箱管理** - 批量采集、编辑标题/价格/SKU，实时利润计算
- 🚫 **违禁词检测** - 138个违禁词库，输入即检，红色高亮提示
- ✨ **AI标题优化** - 接入 Claude API，一键生成6个差异化标题方案
- 💰 **分档加价** - 进货≤10元×3，≤20元×2.5，≤50元×2.2，≤100元×2，≤999元×1.8，自动尾数美化
- 🔗 **链接铺货** - 粘贴1688/淘宝/拼多多链接自动解析商品信息
- 🚀 **一键上架** - 自动打开抖店发布页并填写标题、售价

## 安装方法

1. 下载最新 [plugin.zip](../../releases/latest)
2. 解压到任意文件夹
3. Chrome 打开 `chrome://extensions`
4. 开启「开发者模式」
5. 点「加载已解压的扩展程序」选择解压的文件夹
6. 点插件图标即可打开

## 更新方法

插件内「设置」页面点「检查更新」，发现新版本后点「下载最新版」，
下载后把新 `panel.html` 拖入插件目录的 `panel/` 文件夹覆盖旧文件，
然后在 `chrome://extensions` 点刷新即可。

